# Filmorate - Веб-приложение для оценки фильмов

Учебный проект по созданию социальной сети для оценки фильмов, разработанный в команде из 4 человек.

## О проекте

Filmorate - это платформа для любителей кино, которая позволяет пользователям оценивать фильмы, писать отзывы, добавлять друзей и находить новые фильмы на основе рекомендаций.

## Роль в проекте: Team Lead

В рамках проекта выполнял роль тимлида, отвечая за:
- Распределение задач между 4 разработчиками
- Проведение регулярных код-ревью
- Контроль сроков выполнения задач
- Интеграцию отдельных модулей в единую систему
- Координацию командной работы
- Реализацию ключевых модулей приложения

## Функциональность

### Управление фильмами
- Добавление и редактирование информации о фильмах
- Система оценок и рейтингов
- Категоризация по жанрам и рейтингам
- Поиск фильмов по различным критериям

### Социальные функции
- Система друзей (добавление, подтверждение, удаление)
- Лента активности друзей
- Рекомендации на основе предпочтений друзей

### Отзывы и рецензии
- Написание отзывов о фильмах
- Оценка отзывов

### Поиск и рекомендации
- Расширенный поиск по жанрам, годам, режиссерам
- Персонализированные рекомендации
- Топы фильмов по различным категориям

## Технологический стек

### Backend
- **Java** - основной язык разработки
- **Spring Boot** - фреймворк для создания приложения
- **Spring MVC** - построение REST API
- **JDBC** - работа с реляционной базой данных
- **H2 Database** - база данных для разработки

### Инструменты разработки
- **Maven** - система сборки проекта
- **Git** - система контроля версий
- **REST API** - архитектура веб-сервиса
- **Postman** - тестирование API endpoints
- **JUnit** - модульное тестирование
